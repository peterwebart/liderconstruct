# ADR-0011: URL structure
Status: Accepted

## Context
URLs must be human-readable, SEO-friendly, bilingual, and never expose supplier
taxonomy.

## Decision
`/products/{slug}`, `/category/{section}/{category}`, `/brand/{brand}`,
`/search`. RU prefixed (e.g. `/ru/...`); RO default may omit the prefix. Slugs
are per-locale; categories use our internal Section/Category slugs only.

## Consequences
- Stable, share-friendly URLs; duplicate-name products disambiguated by slug +
  canonical.
