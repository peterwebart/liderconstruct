# ADR-0009: Internationalization — RO default + RU
Status: Accepted

## Context
Catalog is ~98% RO today; RU is the biggest organic-search opportunity.

## Decision
Payload localization with locales RO (default) + RU; localized titles,
descriptions, slugs and SEO. Per-locale URLs with hreflang. Admin defaults to RO.

## Consequences
- RU fields fill in over time (incl. via future AI enrichment, ADR ties to
  enrichment-ready schema). No schema change needed to translate.
